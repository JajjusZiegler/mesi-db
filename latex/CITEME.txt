The R-markdown file <vignettes/cite_mesi.Rmd> can be used create a latex ready table with citations from mesi-db. 

For an example on how it could look you can render example.tex or have a look at example.pdf.

Generate bibliography with Zotero

In order to create the correct bibtex file follow this guideline:

Create zotero entry using information from column `full_reference` in table `data/mesi_bibliography.csv` , in mesi-db repository.

In order to generate citation keys for your Zotero project that match the citation column in `data/mesi_bibliography.csv` follow this:

		1. download better Bibtex plugin for Zotero (https://retorque.re/zotero-better-bibtex/)

		2.Look here for documentation on how to generate bibtex key (https://retorque.re/zotero-better-bibtex/citing/index.html)

			—> working key gernerator string :

			authEtAl("*",false,"*").lower.replace("etal","et_al") + "_*" + year

			edgecases: 

			1. author name includes ä , ö , ü | ä = ae , ö =oe , ü = ue
			2. exactly two authors | add ‘_and’ between the two authors
			3. author name consists of multiple words eg. Van Kulen, De Jong ec. | add ‘_’ between the words

			—> for edge cases use the following expression in the Zotero field ‘extra’ `Citation Key: <your citekey>` <your citekey> 
			should match the `mesi_bibliography.csv`  column `citation` .

			-> Export zotero collection to bibtex (.bib) and make sure that bibtex key is identical to column `citation` in `mesi_bibliography.csv`
